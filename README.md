
# Restaurant Order Processor

This is a coding challenge for redcat technologies

To run the code checkout the master branch


## Run backend Locally

Clone the project


Go to the backend project directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Run frontend Locally

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
